The body contains the content of your page. It's like a document where you
use tags that define the layout, for example: headings, paragraphs, media such
as images and sound, links to other pages and sites, etc.. Here, we use the "p" tag
for paragraph. The "br" tag is for line breaks. Notice how "br" does not have a closing tag.

Headings are a way to add a title and subtitles. There are 6 headings "h1" through "h6" 
where "h1" is most important and "h6" is least important. Margins are automatically
added before and after each heading. Note that the h1 tag is only used once, as the main
heading of the page. h2 to h6 can be used as often as desired, but they should
always be used in order.  Here is an example in action.
