bold vs strong

The "b" tag will bold the content between the tag. This is bold. The "strong"
tag will also bold the text, but has a different meaning on different devices. This is strong.
Bold is a style but has no meaning to a blind person who can't see bold. strong is an indication of how something should be understood.
"Strong" could (and often does) mean "bold" in a browser but it could also mean a lower tone for a speaking program like Jaws (for blind people)
or be represented by an underline (since you can't bold a bold) on a Palm Pilot. You should use strong in favor of bold since it has more meaning.

italic vs emphasis

Similarly "i" will italicize text. This is italicized. "em" or emphasis will also italicize text. This is emphasized.
Where as "i" just means italize text, "em" has additional meaning beyond just seeing italicized text in the browser. "strong" is "emphasis" but stronger. You should use "em" in favor of "i" for its additional meaning across devices.

subscript and superscript

The HTML "sub" tag defines subscripted text. This is subscripted text.
The "sup" tag defines superscripted text. This is superscripted text.

unordered lists

An unordered list starts with the "ul" tag. Each list item starts with the "li" tag. The list items will be marked with bullets by default. Here is an example.

  • apples
  • oranges
  • pears

ordered lists

An ordered list starts with the "ol" tag. Each list item starts with the "li" tag. The list items will be numbered by default. Here is an example.

  1. apples
  2. oranges
  3. pears
